Blockの説明（array<array<array<int[3], 3>, 3>, 3> Blocks;）
・つねに白面をU、緑面をFとする
・配列の座標の決め方：Blocks[上下(U->D)][前後(F->B)][左右(L->R)][3]（0->2）
・(Blocks[i][j][k] [0,1,2])
    [0]=白か黄面の色（0~6）
    [1]=緑か青面の色（0~6）
    [2]=橙か赤面の色（0~6）
色
・Empty = 0;
・White = 1;
・Green = 2;
・Orange = 3;
・Red = 4;
・Blue = 5;
・Yellow = 6;
回転番号
・U = 1;
・U'= 7;
・F = 2;
・F'= 8;
・L = 3;
・L'= 9;
・R = 4;
・R'= 10;
・B = 5;
・B'= 11;
・D = 6;
・D'= 12;
